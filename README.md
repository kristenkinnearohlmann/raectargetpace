# Race Target Pace

A Connect IQ widget to add and display race target pace by distance segment.


Connect IQ SDK: <https://developer.garmin.com/connect-iq/sdk/>

Tutorial from Ottorino Bruni: <https://www.ottorinobruni.com/getting-started-with-garmin-connect-iq-development-build-your-first-watch-face-with-monkey-c-and-vs-code/>

## Running on the Simulator - Mac

- Add to zsh shell ```export PATH=$PATH:`cat $HOME/.Garmin/ConnectIQ/current-sdk.cfg`/bin```

- Palette Build Project

- terminal `connectiq`

- VS Code Run and Debug while the View file is up
